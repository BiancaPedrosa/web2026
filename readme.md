Firebase Authentication Starter

Uma implementação leve e pronta para produção de autenticação de utilizadores utilizando o Firebase v10+. Este projeto demonstra como integrar e gerir sessões de utilizador utilizando dois dos métodos de autenticação mais comuns no mercado: E-mail/Senha e Google OAuth.

🚀 Funcionalidades
Autenticação por E-mail e Senha:

Registo de novos utilizadores (Sign Up).

Login de utilizadores existentes (Sign In).

Fluxo básico para recuperação/redefinição de senha.

Integração com Google OAuth: Login rápido com apenas um clique utilizando contas Google.

Gestão de Estado da Sessão: Monitorização em tempo real do estado de autenticação do utilizador (onAuthStateChanged).

Proteção de Rotas: Estrutura exemplo para bloquear o acesso de utilizadores não autenticados a páginas privadas.

🛠️ Tecnologias Utilizadas
Frontend: HTML5, CSS3, JavaScript (ES6+)

Backend/BaaS: Firebase (Authentication)
